CSS Flexbox e Grid

Uma análise técnica dos sistemas de layout CSS Flexbox e Grid, detalhando suas propriedades, comportamentos e aplicações em contextos de desenvolvimento web contemporâneo.

Definição e Scope

Os modelos de layout Flexbox e Grid representam especificações do CSS projetadas para otimizar a distribuição de elementos em interfaces de usuário, oferecendo controlo preciso sobre alinhamento, ordenação e dimensionamento sem dependência de posicionamentos absolutos ou flutuações.

CSS Flexbox (Flexible Box Layout)

O modelo Flexbox implementa um sistema unidimensional para controlo de layouts em linha ou coluna, otimizado para distribuição de espaço entre items num container, mesmo quando suas dimensões são desconhecidas ou dinâmicas.

Propriedades do container Flex

display: flex | inline-flex

Estabelece um contexto de formatação flexível, transformando o elemento num container flex de nível de bloco ou de linha.

display: flex; /* container flex de nível de bloco */
display: inline-flex; /* container flex de nível de linha */

flex-direction

Define o eixo principal do container, determinando a direção em que os items flex são posicionados.

flex-direction: row; /* Padrão: esquerda para direita */
flex-direction: row-reverse; /* Direita para esquerda */
flex-direction: column; /* Cima para baixo */
flex-direction: column-reverse; /* Baixo para cima */

flex-wrap

Controla o comportamento de transbordo dos items flex quando estes excedem as dimensões do container no eixo principal.

flex-wrap: nowrap; /* Padrão: items numa única linha */
flex-wrap: wrap; /* items quebram para linhas adicionais */
flex-wrap: wrap-reverse; /* items quebram em ordem reversa */

flex-flow

Propriedade abreviada que combina flex-direction e flex-wrap numa única declaração.

flex-flow: row wrap; /* Equivalente a flex-direction: row; flex-wrap: wrap; */

justify-content

Define a distribuição de espaço entre e ao redor dos items flex ao longo do eixo principal do container.

justify-content: flex-start; /* Padrão: items alinhados ao início */
justify-content: flex-end; /* items alinhados ao final */
justify-content: center; /* items centralizados */
justify-content: space-between; /* items distribuídos uniformemente; primeiro ao início, último ao final */
justify-content: space-around; /* items distribuídos com espaço igual ao redor de cada um */
justify-content: space-evenly; /* items distribuídos com espaço igual entre eles */

align-items

Controla o alinhamento dos items flex ao longo do eixo transversal dentro da linha atual.

align-items: stretch; /* Padrão: items esticados para preencher o container */
align-items: flex-start; /* items alinhados ao início do eixo transversal */
align-items: flex-end; /* items alinhados ao final do eixo transversal */
align-items: center; /* items centralizados no eixo transversal */
align-items: baseline; /* items alinhados pela linha de base textual */

align-content

Especifica a distribuição de espaço entre e ao redor das linhas flex quando há espaço extra no eixo transversal.

align-content: stretch; /* Padrão: linhas esticadas para ocupar o espaço disponível */
align-content: flex-start; /* Linhas empacotadas no início */
align-content: flex-end; /* Linhas empacotadas no final */
align-content: center; /* Linhas centralizadas */
align-content: space-between; /* Linhas distribuídas uniformemente */
align-content: space-around; /* Linhas com espaço igual ao redor delas */

Propriedades dos items Flex

order

Controla a sequência de renderização dos items flex, permitindo reordenação visual sem alterar a estrutura do DOM.

order: 0; /* Padrão: posição baseada na ordem do documento */
order: 1; /* Posicionado após items com valores menores */
order: -1; /* Posicionado antes de items com valores maiores */

flex-grow

Define o fator de crescimento de um item flex relativo aos demais items no container.

flex-grow: 0; /* Padrão: não expande além do tamanho base */
flex-grow: 1; /* Expande para ocupar espaço disponível */
flex-grow: 2; /* Expande com prioridade dupla em relação a flex-grow: 1 */

flex-shrink

Estabelece o fator de contração de um item flex quando o espaço disponível é insuficiente.

flex-shrink: 1; /* Padrão: contrai se necessário */
flex-shrink: 0; /* Não contrai mesmo se necessário */
flex-shrink: 2; /* Contrai com prioridade dupla em relação a flex-shrink: 1 */

flex-basis

Especifica o tamanho inicial de um item flex antes da distribuição de espaço disponível.

flex-basis: auto; /* Padrão: baseado nas propriedades de dimensão do item */
flex-basis: 0; /* Tamanho inicial zero, distribuição baseada apenas em flex-grow */
flex-basis: 100px; /* Tamanho inicial específico */

flex

Propriedade abreviada que combina flex-grow, flex-shrink e flex-basis numa única declaração.

flex: 0 1 auto; /* Padrão: não cresce, contrai se necessário, tamanho base auto */
flex: 1; /* Equivalente a flex: 1 1 0 - cresce e contrai uniformemente */
flex: auto; /* Equivalente a flex: 1 1 auto - baseado no conteúdo, mas flexível */
flex: none; /* Equivalente a flex: 0 0 auto - completamente inflexível */

align-self

Substitui o alinhamento padrão para items flex individuais, sobrescrevendo o valor de align-items do container.

align-self: auto; /* Padrão: segue o valor align-items do container */
align-self: flex-start; /* Alinhado ao início do eixo transversal */
align-self: flex-end; /* Alinhado ao final do eixo transversal */
align-self: center; /* Centralizado no eixo transversal */
align-self: stretch; /* Esticado para preencher o container no eixo transversal */

Exemplo de Implementação Flexbox

<div class="flex-container">
  <div class="flex-item">Item 1</div>
  <div class="flex-item">Item 2</div>
  <div class="flex-item">Item 3</div>
</div>
.flex-container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  flex-wrap: wrap;
}

.flex-item {
  flex: 1 1 200px;
  margin: 10px;
}

CSS Grid Layout

O sistema Grid implementa um modelo bidimensional que permite controlo simultâneo sobre linhas e colunas, possibilitando posicionamento preciso e alinhamento de elementos em layouts complexos.

Propriedades do container Grid

display: grid | inline-grid

Estabelece um contexto de formatação grid, transformando o elemento num container grid de nível de bloco ou de linha.

display: grid; /* container grid de nível de bloco */
display: inline-grid; /* container grid de nível de linha */

grid-template-columns e grid-template-rows

Definem as colunas e linhas do grid com uma lista de valores representando o tamanho de cada trilha.

grid-template-columns: 100px 200px 100px; /* Três colunas com tamanhos fixos */
grid-template-rows: auto 200px; /* Duas linhas: primeira automática, segunda fixa */

/* Usando a unidade fr para frações do espaço disponível */
grid-template-columns: 1fr 2fr 1fr; /* Proporção 1:2:1 */

/* Usando repeat() para padrões repetitivos */
grid-template-columns: repeat(3, 1fr); /* Três colunas iguais */
grid-template-columns: repeat(auto-fill, minmax(200px, 1fr)); /* Colunas responsivas */

grid-template-areas

Especifica áreas nomeadas que podem ser referenciadas ao posicionar items no grid.

grid-template-areas:
  "header header header"
  "sidebar content content"
  "footer footer footer";

grid-template

Propriedade abreviada que combina grid-template-rows, grid-template-columns e grid-template-areas numa única declaração.

grid-template:
  "header header header" 100px
  "sidebar content content" auto
  "footer footer footer" 100px
  / 200px 1fr 1fr;

grid-column-gap e grid-row-gap

Especificam o espaçamento entre linhas e colunas do grid.

grid-column-gap: 20px; /* Espaço entre colunas */
grid-row-gap: 15px; /* Espaço entre linhas */

grid-gap

Propriedade abreviada para grid-row-gap e grid-column-gap.

grid-gap: 15px 20px; /* Espaço entre linhas e colunas, respectivamente */
grid-gap: 10px; /* Mesmo espaço para linhas e colunas */

Nota: As propriedades grid-column-gap, grid-row-gap e grid-gap foram renomeadas para column-gap, row-gap e gap, respectivamente, mas as versões prefixadas são mantidas para compatibilidade.

justify-items

Alinha os items do grid horizontalmente dentro de suas células.

justify-items: stretch; /* Padrão: items ocupam toda a largura da célula */
justify-items: start; /* items alinhados ao início da célula */
justify-items: end; /* items alinhados ao final da célula */
justify-items: center; /* items centralizados horizontalmente */

align-items

Alinha os items do grid verticalmente dentro de suas células.

align-items: stretch; /* Padrão: items ocupam toda a altura da célula */
align-items: start; /* items alinhados ao topo da célula */
align-items: end; /* items alinhados ao fundo da célula */
align-items: center; /* items centralizados verticalmente */

place-items

Propriedade abreviada para align-items e justify-items.

place-items: center stretch; /* Centralizado verticalmente, esticado horizontalmente */
place-items: center; /* Centralizado em ambas as direções */

justify-content

Alinha o grid como um todo horizontalmente dentro do container quando o grid é menor que o container.

justify-content: start; /* Padrão: grid alinhado ao início do container */
justify-content: end; /* Grid alinhado ao final do container */
justify-content: center; /* Grid centralizado horizontalmente */
justify-content: space-between; /* Espaço distribuído entre colunas */
justify-content: space-around; /* Espaço distribuído ao redor das colunas */
justify-content: space-evenly; /* Espaço distribuído uniformemente */

align-content

Alinha o grid como um todo verticalmente dentro do container quando o grid é menor que o container.

align-content: start; /* Padrão: grid alinhado ao topo do container */
align-content: end; /* Grid alinhado ao fundo do container */
align-content: center; /* Grid centralizado verticalmente */
align-content: space-between; /* Espaço distribuído entre linhas */
align-content: space-around; /* Espaço distribuído ao redor das linhas */
align-content: space-evenly; /* Espaço distribuído uniformemente */

place-content

Propriedade abreviada para align-content e justify-content.

place-content: center start; /* Centralizado verticalmente, alinhado ao início horizontalmente */
place-content: space-between; /* Espaço distribuído em ambas as direções */

grid-auto-columns e grid-auto-rows

Definem o tamanho das colunas e linhas geradas automaticamente quando items são posicionados fora do grid explicitamente definido.

grid-auto-columns: 100px; /* Colunas implícitas com 100px */
grid-auto-rows: minmax(100px, auto); /* Linhas implícitas com mínimo de 100px */

grid-auto-flow

Controla o algoritmo de posicionamento automático para items não explicitamente posicionados.

grid-auto-flow: row; /* Padrão: preenche por linhas */
grid-auto-flow: column; /* Preenche por colunas */
grid-auto-flow: dense; /* Algoritmo denso que tenta preencher lacunas */
grid-auto-flow: row dense; /* Preenche por linhas com algoritmo denso */

grid

Propriedade abreviada para todas as propriedades explícitas do grid (grid-template-rows, grid-template-columns, grid-template-areas) e implícitas (grid-auto-rows, grid-auto-columns, grid-auto-flow).

grid: 100px 300px / 3fr 1fr; /* Define duas linhas e duas colunas */
grid: auto-flow / 200px 1fr; /* Auto-flow por linhas com duas colunas definidas */
grid: auto-flow dense 100px / 1fr 2fr; /* Auto-flow denso com linhas de 100px e duas colunas */

Propriedades dos items Grid

grid-column-start, grid-column-end, grid-row-start, grid-row-end

Especificam a localização de um item grid dentro do grid definindo as linhas de início e fim do item.

grid-column-start: 2; /* Inicia na linha vertical 2 */
grid-column-end: 4; /* Termina na linha vertical 4 */
grid-row-start: 1; /* Inicia na linha horizontal 1 */
grid-row-end: 3; /* Termina na linha horizontal 3 */

/* Usando palavra-chave span */
grid-column-end: span 2; /* Ocupa duas células horizontalmente */
grid-row-end: span 3; /* Ocupa três células verticalmente */

grid-column e grid-row

Propriedades abreviadas para grid-column-start/grid-column-end e grid-row-start/grid-row-end, respectivamente.

grid-column: 2 / 4; /* De coluna 2 até coluna 4 */
grid-row: 1 / 3; /* De linha 1 até linha 3 */

/* Usando span */
grid-column: 2 / span 2; /* Inicia na coluna 2 e ocupa 2 células */
grid-row: 1 / -1; /* Inicia na linha 1 e vai até a última linha */

grid-area

Especifica o posicionamento e tamanho de um item grid, seja por referência a um nome de área ou definindo as coordenadas de grid.

grid-area: header; /* Posiciona o item na área nomeada "header" */
grid-area: 1 / 2 / 3 / 4; /* Equivalente a grid-row-start: 1; grid-column-start: 2; grid-row-end: 3; grid-column-end: 4; */

justify-self

Alinha um item grid específico horizontalmente dentro de sua célula.

justify-self: stretch; /* Padrão: ocupa toda a largura da célula */
justify-self: start; /* Alinhado ao início da célula */
justify-self: end; /* Alinhado ao final da célula */
justify-self: center; /* Centralizado horizontalmente */

align-self

Alinha um item grid específico verticalmente dentro de sua célula.

align-self: stretch; /* Padrão: ocupa toda a altura da célula */
align-self: start; /* Alinhado ao topo da célula */
align-self: end; /* Alinhado ao fundo da célula */
align-self: center; /* Centralizado verticalmente */

place-self

Propriedade abreviada para align-self e justify-self.

place-self: center end; /* Centralizado verticalmente, alinhado ao final horizontalmente */
place-self: center; /* Centralizado em ambas as direções */

Exemplo de Implementação Grid

<div class="grid-container">
  <header class="header">Cabeçalho</header>
  <nav class="sidebar">Navegação</nav>
  <main class="content">Conteúdo Principal</main>
  <footer class="footer">Rodapé</footer>
</div>
.grid-container {
  display: grid;
  grid-template-columns: 200px 1fr 1fr;
  grid-template-rows: auto 1fr auto;
  grid-template-areas:
    "header header header"
    "sidebar content content"
    "footer footer footer";
  gap: 10px;
  height: 100vh;
}

.header {
  grid-area: header;
}

.sidebar {
  grid-area: sidebar;
}

.content {
  grid-area: content;
}

.footer {
  grid-area: footer;
}

Comparação entre Flexbox e Grid

Casos de Uso Recomendados

Flexbox

  • Layouts unidimensionais (linha ou coluna);
  • Distribuição de espaço entre items de tamanho variável;
  • Alinhamento vertical e horizontal simples;
  • Navegação horizontal ou vertical;
  • Componentes de interface com direção predominante.

Grid

  • Layouts bidimensionais (linhas e colunas simultaneamente);
  • Posicionamento preciso de elementos;
  • Sobreposição intencional de elementos;
  • Layouts complexos com áreas nomeadas;
  • Sistemas de grade responsivos;
  • Layouts assimétricos com alinhamento preciso.

Considerações de Desempenho

Ambos os modelos de layout apresentam desempenho de renderização otimizado em navegadores modernos. A escolha entre eles deve ser orientada pela complexidade do layout e requisitos de alinhamento, não por preocupações de desempenho.

Metodologia de Implementação

A aplicação adequada dos sistemas Flexbox e Grid requer análise da estrutura hierárquica dos elementos e suas relações dimensionais. É recomendada a combinação estratégica de ambos os modelos, utilizando Grid para o layout macro da página e Flexbox para componentes individuais com distribuição linear.
