Funções CSS: Documentação Técnica

As funções CSS são construções sintáticas que permitem a aplicação de valores computados dinamicamente a propriedades de estilo. Esta documentação descreve a implementação e utilização de funções CSS padrão.

Definição e Escopo

Funções CSS são expressões que retornam valores computados em tempo de renderização. Estas expressões são utilizadas em declarações CSS para permitir cálculos, manipulação de valores cromáticos, transformações e outras operações avançadas.

Funções Matemáticas

calc()

Realiza operações aritméticas entre valores dimensionais, permitindo combinações entre unidades incompatíveis.

width: calc(100% - 50px);

min()

Avalia e retorna o valor mínimo dentre os argumentos fornecidos.

width: min(80%, 600px);

max()

Avalia e retorna o valor máximo dentre os argumentos fornecidos.

height: max(300px, 40vh);

clamp()

Limita um valor entre limites inferior e superior, com valor preferencial intermediário.

font-size: clamp(1rem, 2.5vw, 2rem);

Funções de Cor

rgb() e rgba()

Define valores cromáticos utilizando o modelo RGB (Vermelho, Verde, Azul) com parâmetros numéricos.

color: rgb(255, 0, 0); /* Vermelho */
background-color: rgba(0, 0, 255, 0.5); /* Azul com 50% de opacidade */

hsl() e hsla()

Define valores cromáticos utilizando o modelo HSL (Matiz, Saturação, Luminosidade) para maior controle perceptual.

color: hsl(120, 100%, 50%); /* Verde */
background-color: hsla(240, 100%, 50%, 0.3); /* Azul com 30% de opacidade */

lab() e lch()

Especifica cores utilizando espaços cromáticos perceptualmente uniformes, baseados no sistema CIE LAB e LCH.

color: lab(80% 20 40);
background-color: lch(80% 30 45);

color-mix()

Combina duas cores em uma proporção especificada, utilizando um espaço de cores designado.

color: color-mix(in srgb, #34c9eb 25%, white 75%);

Funções de Gradiente

linear-gradient()

Gera uma imagem representando uma transição gradual entre dois ou mais valores cromáticos ao longo de um eixo linear.

background: linear-gradient(to right, red, yellow);

repeating-linear-gradient()

Cria padrões repetitivos de gradientes lineares, definidos por pontos de parada cromáticos.

background: repeating-linear-gradient(45deg, #ccc, #ccc 10px, #fff 10px, #fff 20px);

radial-gradient()

Gera uma imagem representando uma transição gradual entre dois ou mais valores cromáticos que emanam de um ponto central.

background: radial-gradient(circle, red, yellow);

repeating-radial-gradient()

Cria padrões repetitivos de gradientes radiais, definidos por pontos de parada cromáticos concêntricos.

background: repeating-radial-gradient(circle, #ccc, #ccc 10px, #fff 10px, #fff 20px);

conic-gradient()

Produz um gradiente que progride ao redor de um círculo, útil para criação de efeitos circulares como gráficos de pizza.

background: conic-gradient(red, yellow, green, blue, purple);

Funções de Atributo e Variáveis

attr()

Recupera o valor de um atributo do elemento HTML associado e o incorpora como valor de propriedade CSS.

content: attr(data-label);

var()

Implementa referências a propriedades personalizadas (variáveis CSS) definidas com prefixo duplo hífen.

:root {
  --cor-principal: #0077cc;
}
button {
  background-color: var(--cor-principal);
}

env()

Acessa variáveis de ambiente definidas pelo agente de usuário, frequentemente utilizadas para adaptações a características específicas de dispositivos.

padding: env(safe-area-inset-top);

Funções de Referência

url()

Estabelece referência a um recurso externo através de seu identificador uniforme de recursos.

background-image: url('imagens/fundo.jpg');

Funções de Transformação

translate(), translateX(), translateY(), translateZ(), translate3d()

Deslocam elementos no espaço bidimensional ou tridimensional, sem afetar o fluxo do documento.

transform: translate(50px, 20px);
transform: translate3d(50px, 20px, 10px);

scale(), scaleX(), scaleY(), scaleZ(), scale3d()

Modificam as dimensões de um elemento através de fatores de escala nos eixos correspondentes.

transform: scale(1.5);
transform: scale3d(1.5, 0.8, 1);

rotate(), rotateX(), rotateY(), rotateZ(), rotate3d()

Executam rotações em torno dos eixos especificados, expressas em graus, radianos ou rotações completas.

transform: rotate(45deg);
transform: rotateY(0.5turn);

skew(), skewX(), skewY()

Aplicam inclinações não proporcionais a elementos, alterando os ângulos entre eixos horizontais e verticais.

transform: skew(10deg, 20deg);
transform: skewX(30deg);

matrix(), matrix3d()

Implementam transformações através de matrizes de transformação homogêneas, permitindo combinações complexas de efeitos.

transform: matrix(1, 0.2, 0.2, 1, 10, 20);

Funções de Filtro

blur()

Aplica desfoque gaussiano ao elemento, dispersando os valores cromáticos adjacentes.

filter: blur(5px);

brightness()

Ajusta o brilho global do elemento, multiplicando os valores cromáticos pelo fator especificado.

filter: brightness(150%);

contrast()

Modifica o contraste do elemento, ajustando a diferença entre valores claros e escuros.

filter: contrast(200%);

drop-shadow()

Gera uma sombra projetada que segue o contorno alfa do elemento, não limitada a caixas retangulares.

filter: drop-shadow(3px 3px 4px rgba(0, 0, 0, 0.5));

grayscale(), sepia(), invert()

Aplicam efeitos cromáticos predefinidos, convertendo para escala de cinza, tom sépia ou invertendo valores, respectivamente.

filter: grayscale(80%);
filter: sepia(60%);
filter: invert(100%);

Funções de Layout e Grid

fit-content()

Calcula o tamanho de uma caixa baseado no conteúdo intrínseco, limitado por um valor máximo.

width: fit-content(300px);

minmax()

Define um intervalo de tamanho entre valores mínimo e máximo, utilizado em especificações de layout grid.

grid-template-columns: minmax(100px, 1fr) 3fr;

repeat()

Repete um padrão de valores, otimizando a declaração de estruturas regulares em layouts grid.

grid-template-columns: repeat(3, 1fr);

Funções de Contadores

counter() e counters()

Recuperam e formatam valores de contadores CSS, permitindo numeração automática de elementos.

content: counter(section) "." counter(subsection);
content: counters(item, ".") " ";

Funções de Temporização

cubic-bezier()

Define uma função de temporização personalizada através de coordenadas de pontos de controle de uma curva de Bézier cúbica.

transition-timing-function: cubic-bezier(0.42, 0.0, 0.58, 1.0);

steps()

Cria uma função de temporização segmentada que procede em incrementos discretos, útil para animações quadro a quadro.

transition-timing-function: steps(5, end);

Funções Lógicas (Nível Experimental)

toggle()

Implementa alternância condicional entre valores, retornando o primeiro valor aplicável. Especificação em desenvolvimento com suporte limitado.

color: toggle(red, green, blue);

Metodologia de Implementação

A implementação eficaz de funções CSS requer compreensão das interações entre valores computados e o modelo de renderização do navegador. Recomenda-se análise de comportamento entre diferentes agentes de usuário para garantir consistência.
